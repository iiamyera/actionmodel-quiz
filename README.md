# ActionModel Quiz — Vite App

This is a minimal Vite + React scaffold that re-uses your `actionmodel-quiz (1).jsx` component as `App`.

Quick start:

1. From `c:\Users\USER\Downloads\actionmodel-quiz-app` run:

```bash
npm install
npm run dev
```

2. Build for production:

```bash
npm run build
npm run preview
```

Deployment notes:
- If you deploy to Vercel, set the Project Root to `actionmodel-quiz-app` (or point to this folder in your repo).
- If you previously deployed the wrong folder, ensure the platform's root points to this folder so the correct app is built.
